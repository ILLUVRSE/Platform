# Gravity Drop

**Gravity Drop** is a physics-inspired grid puzzle game where you manipulate gravity to slide blocks into goal zones.

## How to Play

1. **Drop Shapes**: Use Left/Right arrows to position the falling shape, and Space to drop it.
2. **Manipulate Gravity**: Change the direction of gravity (G/H or Arrow keys) to slide all blocks on the board.
3. **Score**: Fill the yellow goal tiles with blocks to clear them and score points.
4. **Combos**: Trigger cascades by clearing goals to multiply your score.

## Controls

* **Arrow Left/Right**: Move spawn position
* **Space**: Drop block
* **Arrow Up/Down/Left/Right**: Change Gravity Direction
* **G / H**: Cycle Gravity Clockwise/Counter-Clockwise

## Mobile
* **Tap Left/Right 30% of screen**: Move piece
* **Tap Center**: Drop
* **Tap Top Area**: Cycle Gravity
